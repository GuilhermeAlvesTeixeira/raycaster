# Raycasting Engine

Um motor gráfico baseado em **raycasting**, inspirado em clássicos como *Wolfenstein 3D*.  
Implementado em **C++20** com **SFML**, o projeto renderiza paredes, aplica texturas, iluminação simples e névoa.
